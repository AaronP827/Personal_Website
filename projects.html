Weather App

The weather app was built with Java in Android Studio and allows the user to check the current weather data for a location that they decide. The app gets live data from an API and displays it to the user. There is various data that the app displays such as the current weather, hourly data for up to 48 hours, and the weather for the next couple of days. The app allows the user to swipe down from the top of the screen to refresh the app. The user may also change the units from imperial to metric. The app also uses multiple "activities."
